# Sistema de Gestión de Mascotas

**Autor:** Eunice Belén Malla Coro

##  Descripción del proyecto
Este repositorio contiene dos programas en Python que resuelven el mismo problema: registrar y mostrar la información básica de una mascota.

### 🔹 Programación Tradicional
- Carpeta: `programacion_tradicional/`
- Archivo: `tradicional.py`
- Utiliza funciones y variables.
- Solicita datos por teclado y muestra la información registrada.

### 🔹 Programación Orientada a Objetos (POO)
- Carpeta: `programacion_poo/`
- Archivos: `mascota.py` y `main.py`
- Implementa una clase `Mascota` con atributos y métodos.
- Se crean objetos y se ejecutan sus métodos para mostrar información y sonidos.

##  Reflexión
La **Programación Tradicional** se centra en funciones y variables, lo que resulta sencillo para programas pequeños, pero puede volverse difícil de mantener en sistemas grandes.  
La **Programación Orientada a Objetos (POO)** organiza el código en clases y objetos, permitiendo mayor reutilización, escalabilidad y claridad. Además, refleja mejor el mundo real al modelar entidades con atributos y comportamientos.
